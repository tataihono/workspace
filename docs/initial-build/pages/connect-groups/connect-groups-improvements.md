# Connect Groups Page - Improvement Recommendations

## UX

- Add a listing or carousel of actual connect groups (the Lava templates already support this).
- Show group details: leaders, meeting time, location, capacity.
- Add a "Find a Group" filter (by campus, day of week, group type).
- Add a sign-up or interest form.

## Performance

- If adding group listings, implement pagination or lazy loading for the carousel.

## Accessibility

- Ensure group cards are keyboard navigable if a carousel is added.
- Carousel controls need appropriate ARIA labels.

## SEO

- Target keywords: "small groups Auckland church", "Bible study groups Auckland".
- Add structured data for groups if listings are added.

## Content

- The page is currently very sparse -- add more substance.
- Show what a typical connect group session looks like.
- Add testimonials from group members.
- Explain the progression: Newish Connect -> regular Connect Group.
- List available groups with meeting details (day, time, location, leader).

## Mobile

- Group carousel (if added) should be swipeable with touch support.
- Group details should be easily scannable on small screens.
