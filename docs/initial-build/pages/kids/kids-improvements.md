# Kids Page - Improvement Recommendations

## UX

- Add photos of each program in action (currently only a banner image).
- Add a "What age group is my child?" helper tool.
- Show which campuses offer which programs.
- Add a "Meet our Kids team" section with leader photos and bios.

## Performance

- Simple page with minimal performance concerns.
- Hero image should be optimized (responsive sizes, WebP format).

## Accessibility

- FAQ accordion needs proper ARIA attributes (`aria-expanded`, `aria-controls`, `role="region"`).
- Age ranges should be clear and consistent across programs.
- Safety information should be easy to find and prominent.

## SEO

- Target keywords: "kids church Auckland", "children's ministry Auckland".
- Add FAQ structured data (`FAQPage` schema) for the FAQ section.
- Consider separate landing pages per age group for improved SEO targeting.

## Content

- Add parent testimonials.
- Provide more detail on the safety and vetting process.
- Add a curriculum overview for each program.
- Include photos of actual programs (with parent consent).

## Mobile

- Program cards should stack vertically on small screens.
- FAQ accordion should be touch-friendly with adequate tap targets.
