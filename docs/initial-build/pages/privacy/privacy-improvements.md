# Privacy Policy Page - Improvement Recommendations

## UX

- Add a table of contents at the top for quick navigation to each section
- Display the "Last Updated" date prominently near the top of the page
- Consider expandable/collapsible sections if the policy grows significantly
- Add a "Back to top" link at the end of each section

## Accessibility

- Ensure proper heading hierarchy for screen readers (H1 -> H2 for sections)
- Add `lang` attribute for any legal terminology if needed
- Ensure sufficient contrast for body text
- Use semantic HTML (`<article>`, `<section>`) for proper document structure

## SEO

- Consider adding `noindex` meta tag if the privacy page does not need to rank in search
- Add a proper meta description summarizing the policy
- Use canonical URL to avoid duplicate content issues

## Content Updates Needed

The current policy may need updating for full compliance:

### NZ Privacy Act 2020
- Review against all 13 Information Privacy Principles (IPPs)
- Add specific data retention periods (how long data is kept)
- Add data breach notification process
- Add cross-border data transfer information (if applicable, e.g., cloud services)
- Add right to deletion ("right to be forgotten")

### Cookies and Analytics
- Should mention cookies and analytics tracking currently in use:
  - Google Analytics (G-1R09W3HMNX)
  - New Relic monitoring
- Add a cookie consent mechanism
- Describe what cookies are set, their purpose, and duration

### Data Storage
- Mention the Rock RMS to Payload CMS migration in the data storage section (when applicable)
- Clarify where data is physically hosted (NZ vs overseas servers)
- Describe any third-party services that process personal data (Fundable, Google Forms, etc.)

### Additional Sections to Consider
- Cookie policy (or separate page)
- Third-party services and data sharing
- Data retention schedule
- Children's privacy (if applicable to youth programs)
- Email marketing opt-out process
- Social media data handling

## Legal

- Engage legal counsel to review against NZ Privacy Act 2020 requirements before the rebuilt site launches
- Consider adding a data breach notification process
- Document the complaints process more explicitly
- Add contact details for the NZ Privacy Commissioner as an escalation path
