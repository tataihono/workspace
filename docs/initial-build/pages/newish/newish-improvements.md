# Newish Connect Page - Improvement Recommendations

## UX

- Add upcoming session dates prominently on the page.
- Show what each week covers (week-by-week outline).
- Add testimonials from people who have been through Newish Connect.
- Show the pathway: Newish Connect -> Connect Groups -> deeper involvement.
- Add campus-specific details (when and where Newish runs at each campus).

## Performance

- Simple page with minimal performance concerns.
- Banner image should be optimized.

## Accessibility

- All form fields need proper labels and error messaging.
- Campus dropdown should be accessible with keyboard navigation.
- Phone input should have clear format guidance.

## SEO

- Target keywords: "new to church Auckland", "church membership Auckland".
- Lower priority SEO page since most traffic comes from internal referrals (Connect Groups page, personal invitations).

## Content

- Expand description with what each week covers.
- Add photos from past Newish Connect sessions.
- Explain the connection to Connect Groups (Newish is the gateway).
- Add a timeline or pathway graphic showing the journey from newcomer to connected member.

## Mobile

- Form should be easy to complete on mobile devices.
- Campus selector should be easy to use on small screens.
- Submit button should be full-width on mobile.
