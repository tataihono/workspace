# Youth Page - Improvement Recommendations

## UX

- Add specific location for Friday nights (which campus?).
- Add a calendar/schedule showing upcoming Fridays.
- Show what a typical Friday night looks like (timeline breakdown).
- Add a sign-up or registration option.

## Performance

- Leader photos should be optimized (responsive sizes, WebP format).
- Consider lazy loading below-fold images (leader gallery).

## Accessibility

- All images need proper descriptive alt text.
- Program times should use the `<time>` HTML element for semantic correctness.

## SEO

- Target keywords: "youth group Auckland", "Friday night church youth Auckland".
- Add `Event` structured data for recurring Friday night sessions.

## Content

- Add student testimonials.
- Show photos from actual youth nights in action.
- Clarify which campus(es) youth runs at.
- Add a parent info section covering safety, drop-off/pick-up details.
- Missing information: location/address for Friday nights.

## Mobile

- Program cards should stack vertically on small screens.
- Leader photos should display in a responsive grid layout.
