# Explaining Christianity Page - Improvement Recommendations

## UX

- Add upcoming session dates prominently on the page.
- Show what each of the 4 weeks covers (week-by-week outline).
- Add location information (currently only available on event listings).
- Add a "What to expect" section for first-timers.
- Show testimonials from past participants.

## Performance

- Simple page with minimal performance concerns.
- Banner image should be optimized.

## Accessibility

- Form fields need proper labels and validation messages.
- Clear submission confirmation feedback after form submit.
- Error states should be visually distinct and screen-reader friendly.

## SEO

- Target keywords: "Christianity course Auckland", "explore Christianity Auckland", "Alpha course alternative Auckland".
- Add `Course` structured data markup.

## Content

- Currently sparse -- expand with more detail about each week's topic.
- Add an FAQ section: What if I miss a week? Do I need to bring anything? Is it free?
- Add a video teaser or introduction.
- Show upcoming dates and locations clearly on this page (not just on event listings).

## Mobile

- Form should be easy to complete on mobile with appropriate input types.
- Keep key info (dates, location) above the fold.
- Submit button should be full-width on small screens.
